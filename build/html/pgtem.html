Coordinator class

The class Coordinator contains methods for a “Coordinator”-agent (an instance) to do two things: a) propose possible grid extensions to connect new sites. b) negotiate between Grid Providers and the Regulator instance to find cheapest allowed extension.

Creator class

The class Creator contains methods for a “Creator”-agent (an instance) to suppose new sites for power generation and loads.

This class can either be instantiated as a sink (load) or source (generator).

class pgtem.agents.Provider(type='HV', **kwargs)[source]¶

Provider class

The class Provider contains methods for a “Provider”-agent (an instance) to put price tags on the grid extensions proposed by the Coordinator instance.

This class can be instantiated as an LV, MV or HV (incl. UHV) grid provider.

The role of this agent might be extended in the future.
